# ⚽ Football Passing Network & Tactical Hub Analysis (Graph Theory)

## 📌 Project Overview
In modern football, finding the tactical heartbeat of a team requires more than counting passes; it requires analyzing the network. This Data Analysis project leverages **Graph Theory** to model team passing structures, identifying the most influential players who act as tactical bridges between defense and attack.

## 🚀 Key Analytical Value
* **Network Graph Modeling:** Utilized `NetworkX` to construct a directed passing graph from match event data, mapping players as nodes and passes as weighted edges.
* **Centrality Metrics:** Calculated **Betweenness Centrality** to pinpoint the team's "hidden playmaker" (the player most frequently on the shortest path between other players) and **Degree Centrality** to measure overall involvement.
* **Tactical Visualization:** Designed an interactive, pitch-mapped dashboard using `Plotly` to visualize the passing network, automatically sizing nodes based on their tactical influence.

## 🛠️ Tech Stack
* **Domain:** Sports Analytics, Graph Theory
* **Language:** Python 3.x
* **Libraries:** `NetworkX` (Graph Modeling), `Pandas` (Data Manipulation), `Plotly` (Interactive Visualization)
